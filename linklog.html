𝕏 Import X / Twitter Bookmarks

Uses the X API v2 Bookmarks endpoint. You need a Bearer Token from the X Developer Portal. Enter your @username and the app resolves your ID automatically.

Step 1 — Credentials
Bearer Token (from Developer Portal → Keys & Tokens)
Your @username
⚠ Browser CORS note: X's API blocks direct browser requests. The fetch below is routed through corsproxy.io. If that proxy is unavailable, use the Paste JSON tab instead — run the API call yourself (curl / Postman) and paste the raw response.
Paste API Response JSON

Run this in a terminal (replace TOKEN and UID):
curl -H "Authorization: Bearer TOKEN" \
"https://api.twitter.com/2/users/UID/bookmarks?tweet.fields=created_at,author_id,text&expansions=author_id&max_results=100"
